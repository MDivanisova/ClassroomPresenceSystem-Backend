# Систем за присуство во училница - Backend

ClassroomPresenceSystem е RESTFULL API backend кој овозможува управување со евиденција на присуство на студенти.

## Користени библиотеки

- https://expressjs.com/
   Node.js пакет за креирање на веб апликации и REST API
- https://mongoosejs.com/
   Node.js пакет за работа со MongoDB база на податоци
- https://www.npmjs.com/package/bcrypt
   Node.js пакет за хеширање и верификација на лозинки
- https://www.npmjs.com/package/jsonwebtoken
   Node.js пакет за генерирање и верификација на JWT токени за автентикација
- https://www.npmjs.com/package/connect-mongo
   Node.js пакет за конекција со MongoDB
- https://zod.dev/
   Node.js пакет за валидација на податоци
